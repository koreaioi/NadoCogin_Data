# 정렬

데이터프레임.sort_values([정렬할 컬럼], 정렬조건 )   
ascending = True면 오름차순 (Default)
ascending = False면 내림차순

ascending 또한 [] 안에서 정렬기준을 달리할 수 있음   
ex df.sort_values(['국어','영어'], ascending=[True,False])

import numpy as np
