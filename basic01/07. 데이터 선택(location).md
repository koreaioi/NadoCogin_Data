
```python
df.loc['5번'] # index 5번에 해당하는 전체 데이터를 가져온다
```

```python
# row 전체 데이터가 아닌 row 일부를 가져오고자 할 때
df.loc['1번', '국어'] # index 1번에 해당하는 국어 데이터를 가져온다
```




    90




```python
df.loc[['1번','2번'], '영어'] # index 1번, 2번에 해당하는 영어 데이터
```




    지원번호
    1번    85
    2번    35
    Name: 영어, dtype: int64




```python
df.loc[['1번','2번'],['영어','수학']]
```
